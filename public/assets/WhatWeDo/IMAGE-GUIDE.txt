HOW TO UPDATE THE "WHAT WE DO" IMAGE
=====================================

1. Replace this file with your new image:
   focus-areas.jpg

2. Or add a new image and update the path in:
   data/site-data.ts
   
   In WHAT_WE_DO, change: image: { src: '/assets/WhatWeDo/your-image.jpg', ... }

3. Recommended: Use a photo showing people/community (e.g. 1200x900px)
4. Supported formats: .jpg, .jpeg, .png, .webp
